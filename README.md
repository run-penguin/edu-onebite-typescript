# 타입스크립트가 필요한 이유
- 자바스크립트는 웹 브라우저에서 간단한 상호작용을 위해 탄생 -> 안정성이나 견고함은 일부 포기하고 빠르고 간결하게 만드는 것에 중점
- Node.js의 등장으로 자바스크립트를 어디서든 실행할 수 있게 됨 -> 다양한 프로그램을 만들기 시작 -> 엄격하지 않은 문법 때문에 안정성 부족 -> 타입스크립트 등장
